# asdf-ytt

[![GitHub Actions Status](https://github.com/sylvainmetayer/asdf-ytt/workflows/Main%20workflow/badge.svg?branch=main)](https://github.com/sylvainmetayer/asdf-ytt/actions)

[ytt](https://github.com/vmware-tanzu/carvel-ytt) plugin for [asdf](https://github.com/asdf-vm/asdf) version manager

## Install

Install the plugin:

```
$ asdf plugin add ytt https://github.com/sylvainmetayer/asdf-ytt
$ asdf list-all ytt
$ asdf install ytt 0.31.0
```
